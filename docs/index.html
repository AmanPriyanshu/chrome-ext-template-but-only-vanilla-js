Installing

Node >= 14.18.0

    # with npm
    λ npm create chrome-ext@latest
    # or with pnpm
    λ pnpm create chrome-ext
    # or with yarn
    λ yarn create chrome-ext
    # or with npx
    λ npx create-chrome-ext
